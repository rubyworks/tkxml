TkXML

Homepage | Source Code | Maling List

DESCRIPTION

An XML markup lanaguage for generating Tk user-interfaces. It was inspired by GTK’s glade format. The TkXML format defines a very simple one-to-one correlatation to the Ruby Tk API.

SYNOPSIS

Here’s an example.xml TkXML file:

<Tk:Root name="test" title="Test">
  <Tk:Frame name="menuframe">
    <Tk:Menubutton name="filebutton" text="File" underline="0">
      <Tk:Menu name="filemenu" tearoff="false">
        <Tk:_add _1="command" label="Open" underline="0" accel="Ctrl+O" command="openDocument" />
        <Tk:_add _1="command" label="Exit" underline="0" accel="Ctrl+Q" command="exitApplication" />
      </Tk:Menu>
      <Tk:_pack side="left" />
    </Tk:Menubutton>
    <Tk:_pack side="left" />
  </Tk:Frame>
  <Tk:_bind _1="Control-o" _2="openDocument" />
  <Tk:_bind _1="Control-q" _2="exitApplication" />
</Tk:Root>

To use the library:

require 'tkxml'

tkxml = TkXML.new(File.read('example.xml'))
tkxml.build

STATUS

At ths point, TkXML is little more than an interesting expirement in encoding GUIs via markup. But it certainly has the potential of becoming a productive tool if someone were inclined to further it’s development.
